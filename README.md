## Getting Started 

### Install: 

```bash
git clone git@github.com:MikeG27/Face-Detection.git
cd Face-Detection
pip install -r requirements.txt
```

### Usage:
```bash
To run the script type in 'python face_detector -i [input folder] -o [output folder ]>'
```
